# Formatting Requirements (ETTI / UPB)

Scope: Non-negotiable formatting and submission constraints for the dissertation document and final PDF package.
Source: ETTI "Regulations on the preparation of the diploma/dissertation thesis" (English version).

## 1) Mandatory thesis structure (document-level)
- Title page with the required fields (as defined by the faculty template / annexes).
- Chapters and sections organized logically; chapters are numbered.
- Conclusions section (not numbered as a chapter, depending on template rules).
- Bibliography listing all referenced sources.
- Annexes (when applicable), placed after bibliography.

## 2) Typography and layout (high-level constraints)
- Use a proportional font; recommended serif fonts (e.g., Times New Roman / Computer Modern).
- Font size for body text: 11–12 pt.
- Justified alignment for regular text; captions/titles may be centered according to template.

## 3) Figures, tables, equations
- Every figure/table must have a caption and must be referenced in the text.
- Include sources for externally-derived figures/tables either via bibliography entry or in the caption (when allowed by the regulation).
- Maintain consistent numbering and labeling (LaTeX handles this if you use \\label/\\ref).

## 4) Citations and bibliography rules
- Cite all non-original material (text, figures, ideas, datasets, and code inspiration if relevant).
- Prefer stable, citable sources (books, papers, official documentation).
- For web sources: include access date; include page numbers when referencing a specific page range (when applicable).

## 5) Academic integrity
- No plagiarism: direct quotes must be quoted and cited; paraphrases must also be cited.
- Translation of a source still requires citation.

## 6) Electronic submission packaging (PDF)
- Submit a **single PDF** containing the full thesis from cover to the last annex.
- Include scanned/signed documents inside the PDF at the specified positions:
  - Annex 1/2 (Thesis description / registration form), signed (scanned)
  - Annex 5 (Academic honesty declaration), signed (scanned)
- File naming rule for the final PDF:
  - `FamilyName_Name1[_Name2...].pdf`
  - Example for you: `Lupu_Silviu-George.pdf` (final PDF)

## 7) Template compliance
- Do not modify the faculty `.cls` unless the faculty explicitly requires it.
- Keep the thesis compiling at all times; avoid template-breaking changes.

## Checklist (keep updated)
- [ ] Title page fields complete and correct
- [ ] Body font size 11–12 pt; consistent typography
- [ ] All figures/tables have captions + are referenced in text
- [ ] All external material is cited
- [ ] Single PDF package (cover → last annex)
- [ ] Annex 1/2 scanned + inserted
- [ ] Annex 5 scanned + inserted
- [ ] Final PDF filename matches rule
